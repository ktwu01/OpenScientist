## Fact

ResearchSkills skills are organized by cognitive memory type rather than arbitrary topical categories:

- Procedural memory: IF-THEN decisions at research impasses, grounded in Soar-style impasse types.
- Semantic memory: missing scientific facts or authoritative project knowledge, grounded in ACT-R-style declarative memory.
- Episodic memory: concrete research episodes and turning points, grounded in case-based reasoning.

Procedural subtypes are `tie`, `no-change`, `constraint-failure`, and `operator-fail`.

## Evidence

The scientist supplied this as the project's authoritative internal definition, explicitly grounding procedural skills in Soar, semantic skills in ACT-R, and episodic skills in case-based reasoning.

## Expiry Signal

Expires if the paper or schema abandons the Soar / ACT-R / CBR framing, changes the procedural / semantic / episodic taxonomy, or reorganizes skills primarily by scientific domain rather than cognitive retrieval mode.
