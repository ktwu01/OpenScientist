## Situation
An AI review loop was first run on a draft section using an inferred notion of "physics mechanisms." The user then clarified that there was already an explicit rule defining the intended scientific standard, and that the rule text itself should not be reviewed.

## Action
The review process was reoriented: the explicit physics-mechanisms rule was treated as the evaluation rubric and applied directly to the manuscript. The reviewer then inspected the draft only for rule violations.

## Outcome
This immediately surfaced concrete methodological problems that the generic review loop had not resolved cleanly, including leaderboard-style table framing and overgeneralized claims that one model class achieves better metrics. Revising those passages produced a draft judged compliant with the intended standard.

## Lesson
When a scientific writing standard already exists as an explicit rubric, use it as the operative review instrument. Generic "improve rigor" review is too underconstrained and tends to miss rubric-specific violations.

## Retrieval Cues
- A user says the draft should "follow this rule" or "apply this standard."
- Review feedback feels stylistic but not methodologically aligned.
- The manuscript has lingering broad superiority claims or ranking language despite prior review.
